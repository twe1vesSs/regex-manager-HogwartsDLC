# 🍏 Regex Manager

Менеджер регексов для [**Яблочного пресета**](https://kykaaj.github.io/yablochny/) в SillyTavern.

## Установка

1. Откройте SillyTavern
2. Перейдите в **Расширения** → **Установить расширение**
3. Вставьте ссылку:https://github.com/kykaaj/regex-manager
4. Нажмите **Установить**
5. Перезагрузите SillyTavern

## Паки регексов

- **hide reasoning** — скрывает блоки размышлений модели
- **html vanquisher** — удаляет HTML из промпта
- **braille blank jb** — брайль-пробелы для JB
- **clocks** — красивые часы с веточками
- **transitions** — переходы между сценами
- **music player** — плеер с бегущей строкой
- **infoblock** — сводка с персонажами и отношениями

## Использование

1. Найдите **🍏 Regex Manager** в настройках расширений
2. Включите нужные паки галочками
3. Готово!
